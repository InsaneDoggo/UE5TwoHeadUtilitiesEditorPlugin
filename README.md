## Installation

1. Download .zip from releases page
2. Put content into `<Your Project>/Plugins/` dir
3. Setup hotkeys: Editor Settings -> search for "TwoHead"
	
![Hotkey](Docs/Images/Hotkey.png)

4. Some behavior can be adjusted via dedicated Editor Settings Page:

![EditorSettingsPage](Docs/Images/EditorSettingsPage.png)
